Florida COVID-19 Data

Parsed from the Florida Department of Health COVID-19 status page.

FL DOH Main Page

The table from the FL DOH main page is parsed and saved verbatim into <covid-19-florida-cases.csv>. The as of timestamp listed on the web page is added into the table in the timestamp column.

screenshot

FL DOH Dashboard

Two tables are extracted from the Floridas COVID-19 Data and Surveillance Dashboard.

  • County-level case count: <covid-19-florida-cases-county.csv>
  • Current test counts: <covid-19-florida-tests.csv>

The last updated timestamp is also extracted from the dashboard and saved in the timestamp column of both of the above tables.
